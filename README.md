# AutoGo - Automotive E-Commerce & Service Appointment System
ระบบสั่งซื้อยางรถยนต์ออนไลน์และจองคิวเปลี่ยนยาง

## Project Overview
AutoGo เป็นแพลตฟอร์ม E-Commerce เต็มรูปแบบที่ออกแบบมาเพื่อธุรกิจยางรถยนต์ โดยรวมประสบการณ์การช้อปปิ้งออนไลน์เข้ากับการนัดหมายบริการหน้าสาขา ลูกค้าสามารถเลือกซื้อยางที่ตรงรุ่น ชำระเงินผ่านระบบ QR Code และเลือกวันเวลาเพื่อเข้ารับบริการเปลี่ยนยางที่ศูนย์บริการได้

## Tech Stack
* **Framework:** Laravel 5.7 (PHP)
* **Database:** MySQL
* **Payment Integration:** QR Code Payment Gateway (PromptPay / Bank Integration)
* **Frontend:** Bootstrap, jQuery, AJAX
* **Key Features:** * Product Catalog & Search Filter (ระบบเลือกซื้อตามขนาดยาง)
    * Shopping Cart & Checkout (ระบบตะกร้าสินค้า)
    * QR Code Payment (ระบบชำระเงินออนไลน์)
    * Service Scheduling (ระบบจองคิวเข้าหน้าสาขา)

## Key Features & Logic Highlights

* **E-Commerce Workflow:** ระบบเลือกซื้อสินค้าที่รองรับเงื่อนไขเฉพาะของยางรถยนต์ (ยี่ห้อ, ขนาด, รุ่น) พร้อมระบบคำนวณราคาสุทธิ
* **Seamless Payment Integration:** พัฒนาระบบชำระเงินผ่าน QR Code ที่ปลอดภัยและใช้งานง่าย ช่วยให้ปิดการขายได้รวดเร็วผ่านมือถือ
* **O2O (Online to Offline) Service:** ระบบจองคิวรับบริการหลังจากชำระเงินเสร็จสิ้น โดยลูกค้าสามารถเลือกสาขาและเวลาที่สะดวก
